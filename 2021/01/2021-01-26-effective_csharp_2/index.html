мқҙнҺҷнӢ°лёҢ c#

м•„мқҙн…ң 5: л¬ёнҷ”к¶Ңлі„лЎң лӢӨлҘё л¬ёмһҗм—ҙмқ„ мғқм„ұн•ҳл Өл©ҙ FormattableStringмқ„ мӮ¬мҡ©н•ҳлқј

м•„мқҙн…ң 6: nameof() м—°мӮ°мһҗлҘј м Ғк·№ нҷңмҡ©н•ҳлқј

nameof мӢқмқҖ ліҖмҲҳ, нҳ•мӢқ лҳҗлҠ” л©ӨлІ„мқҳ мқҙлҰ„мқ„ л¬ёмһҗм—ҙ мғҒмҲҳлЎң к°Җм ёмҳөлӢҲлӢӨ.

nameofлҘј нҷңмҡ©н•ҳл©ҙ мқёмҲҳ кІҖмӮ¬ мҪ”л“ңлҘј мўҖлҚ” мүҪкІҢ мң м§Җ н• мҲҳ мһҲлӢӨ.

nameof м—°мӮ°мһҗлҘј нҷңмҡ©н•ҳл©ҙ мӢ¬ліјмқҳ мқҙлҰ„мқ„ л°”кҫёкұ°лӮҳ мҲҳм •н•ҳлҠ” кІҪмҡ°м—җлҸ„ мҶҗмүҪкІҢ ліҖкІҪмӮ¬н•ӯмқ„ л°ҳмҳҒн• мҲҳ мһҲлӢӨ.

м•„мқҙн…ң 7: лҚёлҰ¬кІҢмқҙнҠёлҘј мқҙмҡ©н•ҳм—¬ мҪңл°ұмқ„ н‘ңнҳ„н•ҳлқј

Predicate<T>, Action<>, Func<> кіј к°ҷмқҖнҳ•нғңлЎң мһҗмЈјм“°лҠ” лҚёлҰ¬кІҢмқҙнҠёлҘј м •мқҳн•ҙ лҶ“кі  мһҲлӢӨ.

  • Action<> : л§Өк°ң ліҖмҲҳ(T16к№Ңм§Җ мЎҙмһ¬н•Ё)к°Җ мһҲкі  к°’мқ„ л°ҳнҷҳн•ҳм§Җ м•ҠлҠ” л©”м„ңл“ңлҘј мәЎмҠҗнҷ”н•©лӢҲлӢӨ.
  • Predicate<T> : л§Өк°ң ліҖмҲҳк°Җ н•ҳлӮҳмқҙкі  bool к°’мқ„ л°ҳнҷҳн•ҳлҠ” л©”м„ңл“ң
  • Func<> : л§Өк°ңліҖмҲҳк°Җ м—Ҷкұ°лӮҳ мЎҙмһ¬н•ҳл©° TResult нғҖмһ…мқ„ л°ҳнҷҳн•ҳлҠ” л©”м„ңл“ң

LINQлҠ” мң„мҷҖ к°ҷмқҖ к°ңл…җмқ„ кё°л°ҳмңјлЎң л§Ңл“Өм–ҙ мЎҢлӢӨ.

лӘЁл“  лҚёлҰ¬кІҢмқҙнҠёлҠ” кё°ліём ҒмңјлЎң л©ҖнӢ°мәҗмҠӨнҠёк°Җ к°ҖлҠҘн•ҳлӢӨ.
л©ҖнӢ°мәҗмҠӨнҠё лҚёлҰ¬кІҢмқҙнҠёлҠ” н•ң лІҲл§Ң нҳём¶ңн•ҳл©ҙ лҚёлҰ¬кІҢмқҙнҠё к°қмІҙм—җ м¶”к°Җлҗң лӘЁл“  лҢҖмғҒ н•ЁмҲҳк°Җ нҳём¶ңлҗңлӢӨ.

мң„мҷҖ к°ҷмқҖ кө¬мЎ°м—җлҠ” 2к°Җм§Җ мЈјмқҳ мӮ¬н•ӯмқҙ мһҲлӢӨ.

  • мҳҲмҷём—җ м•Ҳм „н•ҳм§Җ м•ҠлӢӨ.
  • л§Ҳм§Җл§үмңјлЎң нҳём¶ңлҗң лҢҖмғҒ н•ЁмҲҳмқҳ л°ҳнҷҳк°’мқҙ лҚёлҰ¬кІҢмқҙнҠёмқҳ л°ҳнҷҳк°’мңјлЎң к°„мЈјлҗңлӢӨ.

лҚёлҰ¬кІҢмқҙнҠёлҠ” лҹ°нғҖмһ„м—җ мҪңл°ұмқ„ кө¬м„ұн•ҳлҠ” мөңкі мқҳ л°©лІ•мқҙлӢӨ.

м•„мқҙн…ң 8: мқҙлІӨнҠё нҳём¶ң мӢңм—җлҠ” null мЎ°кұҙ м—°мӮ°мһҗлҘј мӮ¬мҡ©н•ҳлқј

мң„м—җ мҪ”л“ңм—җм„ң Updated к°’мқҙ null мқҙл©ҙ NullReferenceExceptionмқҙ л°ңмғқн•ңлӢӨ.

к°„лӢЁн•ҳкІҢ мҲҳм •н•ҙліҙл©ҙ null мІҙнҒ¬лҘј л„Јм–ҙ мЈјл©ҙ лҗңлӢӨ мң„м—җм„ңлҸ„ мҳӨлҘҳк°Җ л°ңмғқн• мҲҳ мһҲлҠ”лҚ° л©ҖнӢ° м“°л Ҳл“ң нҷҳкІҪм—җм„ң null мІҙнҒ¬лҘј нҶөкіјн•ң
лҸҷмӢңм—җ лӢӨлҘё м“°л Ҳл“ңм—җм„ң Updated л ҲнҚјлҹ°мҠӨлҘј null лЎң м№ҳнҷҳ н•ҙлІ„лҰ¬л©ҙ лҳҗ мҳӨлҘҳк°Җ лӮ мҲҳмһҲлӢӨ.

мң„м—җ мҪ”л“ңк°Җ c#м—җм„ң к¶ҢмһҘн•ҳлҠ” мҪ”л“ңмқҙлӢӨ.

null мЎ°кұҙ м—°мӮ°мһҗлҘј нҶөн•ҙм„ң к°„лӢЁн•ҳкІҢ ліҖкІҪмӢңнӮЁ мҪ”л“ңмқҙлӢӨ.

c# м»ҙнҢҢмқјлҹ¬лҠ” лӘЁл“  лҚёлҰ¬кІҢмқҙнҠёмҷҖ мқҙлІӨнҠём—җ лҢҖн•ҙ Invoke л©”м„ңл“ңлҘј нғҖмһ… м•Ҳм •м Ғ нҳ•нғңлЎң мғқм„ұн•ҙ мЈјлҜҖлЎң мқҙ л©”мҶҢл“ңлҘј нҳём¶ңн•ҳлҠ” кІғмқҖ ()лҘј мқҙмҡ©н•ҳм—¬ мқҙлІӨнҠёлҘј л°ңмғқмӢңнӮӨлҠ” мҪ”л“ңмҷҖ к°ҷмқҖ мҪ”л“ңмқҙлӢӨ.

м•„мқҙн…ң 9: л°•мӢұкіј м–ёл°•мӢұмқ„ мөңмҶҢнҷ”н•ҳлқј

л°•мӢұ : к°’ нғҖмһ…мқҳ к°қмІҙлҘј нғҖмһ…мқҙ м •н•ҙм ё мһҲм§Җ м•ҠмқҖ мһ„мқҳмқҳ м°ёмЎ° нғҖмһ… лӮҙл¶Җм—җ нҸ¬н•ЁмӢңнӮӨлҠ” л°©лІ•
м–ёл°•мӢұ : л°•мӢұлҗҳм–ҙ мһҲлҠ” м°ёмЎ° нғҖмһ…мқҳ к°қмІҙлЎңл¶Җн„° к°’ нғҖмһ… к°қмІҙмқҳ ліөмӮ¬ліёмқ„ к°Җм§Җкі  мҳӨлҠ” л°©лІ•

л°•мӢұкіј м–ёл°•мӢұмқҖ м„ұлҠҘм—җ мўӢм§Җ м•ҠмқҖ мҳҒн–Ҙмқ„ лҜём№ңлӢӨ. л•ҢлЎңлҠ” л°•мӢұкіј м–ёл°•мӢұмқҙ мҲҳн–үлҗҳлҠ” кіјм •м—җм„ң мһ„мӢң к°қмІҙк°Җ мғқм„ұлҗҳкё°лҸ„ н•ҳлҠ”лҚ°, к°„нҳ№ мқҙлЎң мқён•ҙ мҳҲмғҒм№ҳ лӘ»н•ң лІ„к·ёк°Җ л°ңмғқн•ҳкё°лҸ„ н•ңлӢӨ.

лҢҖл¶Җл¶„мқҳ кІҪмҡ° м ңл„ӨлҰӯ нҒҙлһҳмҠӨмҷҖ м ңл„ӨлҰӯ л©”м„ңл“ңлҘј мӮ¬мҡ©н•ҳл©ҙ л°•мӢұкіј м–ёл°•мӢұмқ„ н”јн•  мҲҳ мһҲлӢӨ.
