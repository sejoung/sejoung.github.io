    GraphQLмқ„ кіөл¶Җн•ҳл©ҙм„ң лҠҗлӮҖм җ

    SOAP, REST, GraphQL

    м§ҖкёҲк№Ңм§Җ к°ңл°ңмқ„ н•ҙмҳӨл©ҙм„ң 3к°Җм§Җ API м•„нӮӨн…ҚміҗлҘј мӮ¬мҡ©н•ҙмҷ”лӢӨ.

    мІҳмқҢм—җ м•„л¬ҙкІғлҸ„ лӘЁлҘҙлҠ”мІҙ SOAPмқҙлқјлҠ” л°©мӢқмқ„ мӮ¬мҡ©н•ҙм„ң к°ңл°ңмқ„ н–Ҳм—ҲлҠ”лҚ°
    SOAPмқҖ httpлҘј мӮ¬мҡ©н•ҳм—¬ XMLлЎң лҚ°мқҙн„°лҘј мЈјкі  л°ӣм•ҳлӢӨ. м—¬кё°м„ң л©”мҶҢл“ңлҠ” getкіј postл§Ң мӮ¬мҡ©н–ҲмқҢ

    SOAPмқҖ лҢҖнҳ• лІӨлҚ”(?) IBM, ORACLE л“ұ м—җм„ң SOA мӮ¬мғҒмқ„ кө¬нҳ„ н•  мҲҳмһҲлҠ” мҲҳлӢЁмңјлЎң л°Җл©ҙм„ң мІҳлҰ¬лҘј н–ҲлӢӨ.
    м—¬кё°м„ң м—¬лҹ¬к°Җм§Җк°Җ мһҲлҠ”лҚ° SOAPлҸ„ кі„м•Ҫмқҳ мқҳн•ң м„Өкі„к°Җ к°ҖлҠҘн•ҳлӢӨ к·ёкІғмқҖ end pointм—җм„ң wsdlмқ„ л…ём¶ңн•ҳл©ҙм„ң н•ҙлӢ№ м„ңл№„мҠӨмқҳ input, output нғҖмһ…мқ„ л…ём¶ңн•ҳл©ҙм„ң к°ҖлҠҘн–ҲлӢӨ.

    SOAPм—җм„ң RESTлЎң л„ҳм–ҙк°„кІғмқҙ SOAPмқҖ м–ҙл өлӢӨлҠ” мқҙм•јкё°к°Җ л§Һм•ҳлӢӨ. к·ёлҹ° мқҙмң к°Җ XML,XSDл“ұмқ„ кіөл¶Җн•ҙм•ј н• кІғмқҙ л§Һм•„м„ң к·ёлһ¬лҚҳкұ° к°ҷлӢӨ.
    к·ёлҹ¬л©ҙм„ң RESTлЎң л„ҳм–ҙк°”лҠ”лҚ° м§ҖкёҲ лӘҮл…„м „к№Ңм§Җл§Ң н•ҙлҸ„ к·ёлҹ°кІғмқ„ лӘ»лҠҗкјҲлҠ”лҚ° мқҙм   мҷ„м „нһҲ RESTлҠ” н‘ңмӨҖ(?)к°ҷмқҖ лҠҗлӮҢмқҙлӢӨ.

    RESTмқҳ лӢЁм җ(?)мқҖ мІ«лІҲм§ё кі„м•Ҫм—җ мқҳн•ң м„Өкі„к°Җ нһҳл“ӨлӢӨ мҡ”мҰҳм—җлҠ”(мҳӨлһҳм „л¶Җн„° лӮҳмҷ”лҚҳкұ° к°ҷмқҖлҚ°вҖҰ) swagger, wadl л“ұмқ„ нҷңмҡ©н•ҳм—¬ мІҳлҰ¬н• л Өкі  н•ҳлҠ”кІғ к°ҷлӢӨ.
    лҳҗ лҰ¬мҶҢмҠӨ лӢЁмң„лЎң мЎ°нҡҢлҘј н•ҳлӢӨ ліҙлӢҲ overfetching кіј underfetching мқҙ мқјм–ҙлӮҳкі  мһҲлӢӨ.

    overfetchingмқҖ нҒҙлқјмқҙм–ёнҠёк°Җ мӢӨм ңлЎң нҺҳм№ҳ н•  л•Ң н•„мҡ”н•ҳм§Җ м•ҠмқҖ лҚ°мқҙн„°лҘј кІҖмғү мӨ‘мһ„мқ„ мқҳлҜё
    л”°лқјм„ң м•ұмқҳ м„ұлҠҘмқҙ л–Ём–ҙм§Җкі  (лҚ” л§ҺмқҖ лҚ°мқҙн„°лҘј лӢӨмҡҙлЎңл“ңн•ҳкі  кө¬л¬ё л¶„м„қн•ҳлҠ” лҚ° лҚ” мҳӨлһҳ н•„мҡ”) мӮ¬мҡ©мһҗмқҳ лҚ°мқҙн„°лҸ„ л§Һмқҙ мҶҢм§„н•ңлӢӨ.

    underfetchingмқҖ API мқ‘лӢөм—җ м¶©л¶„н•ң лҚ°мқҙн„°к°Җ нҸ¬н•Ёлҗҳм–ҙ мһҲм§Җ м•ҠмқҢмқ„ мқҳлҜё к·ёлһҳм„ң нҒҙлқјмқҙм–ёнҠёк°Җ нҳ„мһ¬ лҚ°мқҙн„° мҡ”кө¬ мӮ¬н•ӯмқ„ м¶©мЎұмӢңнӮӨкё° мң„н•ҙ м¶”к°Җ API мҡ”мІӯмқ„н•ҙм•ј н•Ё
    мқҙ л¬ём ңлҘј н•ҙкІ°н•ҳкё° мң„н•ҙ нҺҳмқҙлЎңл“ңлҘј мҲҳм •н•ҳлҠ” кІҪмҡ°лҸ„ мһҲм§Җл§Ң м•ұ кіј л°ұм—”л“ңк°Җ м—…лҚ°мқҙнҠёлҘј мң„н•ҙ к°ҷмқҙ мӣҖм§Ғм—¬м•ј н•ңлӢӨ.

    мң„ к°ҷмқҖ л¬ём ңк°Җ мғқкё°л©ҙм„ң м§ҖкёҲ GraphQLмқҙ лҢҖм•ҲмңјлЎң л– мҳӨлҘҙкі  мһҲлҠ”кІғ к°ҷлӢӨ.

    GraphQLкіј SOAPмқҳ мң мӮ¬м җмқҖ л‘җ н”„лЎңнҶ мҪң лӘЁл‘җ лҚ°мқҙн„° мң нҳ•мқ„ лӘ…мӢң м ҒмңјлЎң м„ м–ён•ҳлҜҖлЎң мқ‘мҡ© н”„лЎңк·ёлһЁ к°„мқҳ нҶөмӢ мқҙ лӘ…нҷ•н•ҳлӢӨ.

    к·ёлҰ¬кі  л‘ҳлӢӨ лӢЁмқј end point urlмқ„ мӮ¬мҡ©н•ңлӢӨ.

    к·ёлҰ¬кі  GraphQLмқҙлқјлҠ” лӢЁм–ҙм—җм„ң ліҙл“Ҝмқҙ мҝјлҰ¬лҘј м§Җмӣҗн•ңлӢӨ. к·ёлҰ¬кі  лҚ°мқҙн„° ліҖкІҪмқ„ мң„н•ң mutations лҳҗ л°ҳмқ‘нҳ• кө¬лҸ…мқ„ мң„н•ҙм„ң subscriptionмқ„ м§Җмӣҗн•ңлӢӨ.
