мҷң Spring serviceм—җ InterfaceлҘј л§Ңл“Өм–ҙм•ј н• к№Ң?
2017-01-09|java|

Service л¶Җл¶„м—җ interfaceлҘј мӮ¬мҡ©н•ҳлҠ” мқҙмң лҠ” лӢЁмң„ м»ҙнҸ¬л„ҢнҠёлЎңм„ң кІҪкі„к°Җ лҗҳлҠ” л¶Җл¶„мқҙл©°, transaction мІҳлҰ¬, exception мІҳлҰ¬ л“ұмқҳ AOP мІҳлҰ¬к°Җ мЈјлЎң service л¶Җл¶„м—җ м§Җм •лҗҳкё° л•Ңл¬ё(?)

  1. Spring AOPлҠ” л‘җк°Җм§Җ Typeмқҳ ProxyлҘј м§Җмӣҗ к·ё мІ«лІҲм§ёлҠ” JDKмқҳ Proxy кё°лҠҘмқ„ мқҙмҡ© л‘җлІҲм§ё л°©лІ•мқҖ CGLIBмқҳ Enhancer нҒҙлһҳмҠӨлҘј мқҙмҡ©
  2. JDK ProxyлҠ” мқён„°нҺҳмқҙмҠӨм—җ лҢҖн•ң Proxyл§Ңмқ„ м§Җмӣҗ CGLIBмқҳ Enhancer нҒҙлһҳмҠӨлҘј мӮ¬мҡ©н•ҙм„ң л§Ңл“Өл©ҙ лҗҳлҠ”кІғ к·ёлһҳм„ң мң„мқҳ мқҙмң лЎңлҠ” м„ӨлӘ…мқҙ м•ҲлҗЁ
  • к°•м ңлЎң CGLIB нҶөн•ң н”„лЎқмӢңк°қмІҙ мғқм„ұл°©лІ•
  • aop:config нғңк·ём—җ лӢӨмқҢ мҶҚм„ұ м¶”к°Җ : proxy-target-class=вҖқtrueвҖқ
  • м–ҙл…ён…Ңмқҙм…ҳмқҳ кІҪмҡ°

  • egovframeмқҳ лӢөліҖ

  • мһҗл°”м§Җкё°лӢҳмқҳ мӮ¬мқҙнҠё м„ұлҠҘмёЎм • лӮҙмҡ©

interfaceлҠ” м–ҙлҠҗ мӢңм җм—җ мғқм„ұн•ҳлҠ” кІғмқҙ м Ғм Ҳн•ңк°Җ?

вҖңinterfaceлҘј л§Ңл“ңлҠ” кІғм—җ лҢҖн•ң кұ°л¶Җк°җмқҖ мҠӨн”„л§Ғмқҙ л“ұмһҘн•ҳл©ҙм„ң м§ҖмӢқмқ„ м „нҢҢн•  л•Ң лҢҖл¶Җл¶„мқҳ мұ…кіј кҙҖл Ё л¬ём„ңм—җм„ң ServiceмҷҖ Daoм—җлҠ” interfaceлҘј л°ҳл“ңмӢң л§Ңл“ңлҠ” л°©мӢқмңјлЎң мҳҲм ң мҶҢмҠӨлҘј л§Ңл“ӨлӢӨліҙлӢҲ нҳ„мһ¬ лҢҖл¶Җл¶„мқҳ н”„лЎңм қнҠём—җм„ң interfaceлҘј мҷң л§Ңл“Өм–ҙм•ј лҗҳлҠ”м§ҖлҸ„ лӘЁлҘҙлҠ” мғҒнғңм—җм„ң interfaceлҘј л§Ңл“ңлҠ” мғҒнҷ©мқҙ мғқкё°лӢӨ ліҙлӢҲ л°ңмғқн•ң кІғвҖқ

мһҗл°”м§Җкё°лӢҳмқҳ кёҖм—җм„ң ліҙл“Ҝмқҙ лӮҳлҸ„ м•„л¬ҙ мғқк°Ғм—Ҷмқҙ л§Ңл“Өм–ҙм„ң мқҙмҡ©мқ„ н–Ҳм—ҲлҚҳкІғ к°ҷлӢӨ мһҗл°”м§Җкё°лӢҳмқҳ нҸ¬мҠӨнҢ…м—җлҸ„ лӮҳмҷҖмһҲл“Ҝмқҙ лӮҳлҸ„ н•„мҡ”н•ңмӢңм җм—җ л§Ңл“Өм–ҙм„ң м“°лҠ”кІғмқҙ мўӢмқ„кІғ к°ҷлӢӨ.

мқён„°нҺҳмқҙмҠӨк°Җ н•„мҡ”н•ң мқҙмң лҘј мҠӨмҠӨлЎңк°Җ м„ӨлӘ…н•  мҲҳ мһҲкі , к·ёлЎң мқён•ҙ м–»лҠ” мқҙл“қмқ„ м •нҷ•нһҲ м•Ң мҲҳ мһҲмқ„ л•Ң лҸ„мһ…н•ҳлҠ” кІғмқҙ мўӢлӢӨкі  мғқк°Ғн•ңлӢӨ.

мқҙкІғмқҖ лӮҙ мқҳкІ¬мқҙкі  м§ҖмқёмӨ‘ н•ңлӘ…мқҖ мғқм„ұнӣ„м—җ(нҷ•мһҘмқҙ лҚ”мқҙмғҒ м•ҲлҗңлӢӨкі  ліҙмһҘн• мҲҳ м—ҶмқҢлҜҖлЎң) лЁјм Җ мғқм„ұн•ңлӢӨлҠ” мӮ¬лһҢлҸ„ мһҲлӢӨ.

мқҙлҹ° кі лҜјмқ„ н•ңлІҲмҜӨ н•ҙліҙлҠ” кІғлҸ„ мўӢмқ„кІғ к°ҷлӢӨ.
